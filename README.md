# Python Racing Game

## Overview
A 2D racing game built in Python where the player navigates through obstacles, avoids or destroys enemies, and aims to reach the finish line while achieving the highest score.

## Features
- Player-controlled movement across a road
- Shooting mechanic to destroy enemy obstacles
- Enemy obstacles with timed movement patterns
- Collision detection and game-over logic
- Score tracking system
- Integrated leaderboard system using file-based database storage

## Implementation Details
The game was developed using Python and Pygame, applying fundamental programming concepts such as loops, conditionals, and functions.

Enemy behavior and obstacle timing were implemented using Python logic and the Pygame library. The leaderboard system reuses file I/O techniques from a separate database project (Graphics database, also linked in github) to store and retrieve high scores.

## Technologies Used
- Python
- Pygame
- File I/O (for leaderboard storage)

## Purpose
This project demonstrates the application of programming fundamentals to build an interactive real-time game with data storage and user progression features.
